# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-14

### Added
- Initial release of OneID PHP Client
- Core API client functionality with HTTP methods (GET, POST, PUT, DELETE)
- Configuration management with fluent interface
- Authentication methods (login, verify, refresh, logout)
- User profile management
- Custom exception classes for better error handling
- PSR-4 autoloading support
- Comprehensive documentation and examples
- Support for PHP 7.4+

### Features
- Simple and intuitive API interface
- Built-in timeout and debug mode configuration
- Automatic JSON encoding/decoding
- Bearer token authentication support
- Query parameter handling for GET requests
- Error handling with specific exception types
- Configurable base URL and timeouts

### Documentation
- Complete README with usage examples
- Integration guides for oneid-delivery and oneid-portal
- Example usage file with 8 comprehensive examples
- API method reference
- Configuration options documentation

### Developer Experience
- Clean and maintainable code structure
- Proper namespacing with PSR-4
- Type hints and return types
- Detailed inline documentation
- Easy integration with Composer
